# B站自动查成分脚本

这是一个基于油猴（Tampermonkey）开发的浏览器脚本，旨在优化 B 站新版评论区的用户属性标注。它通过穿透 Shadow DOM 获取 UID，并调用 B 站动态接口进行关键词比对。

## 🛠️ 配置说明

你可以在代码顶部的 `captor` 数组中自定义你的关键词规则：

```javascript
// 格式：["显示名称", "关键词1&关键词2|排除词", "颜色"]
const myRules = [
    ["老玩家", "原神&等级>60", "#FFCC00"],
    ["抽奖分母", "互动抽奖", "#254680"]
];
```
